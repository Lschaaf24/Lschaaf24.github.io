Dungeon Crawler Side Project

This is a side project made in Unity that I have been working on in between projects and work. I have worked solo on every aspect of this project from the systems and programming to the animations and art. Some of the highlights from this project include a procedural dungeon generation system following a Room First BSP (Binary Space Partitioning) approach, an enemy pathfinding system, and a combat system currently working with one enemy with abilities such as rolling and parrying.

3D Interactive Environment

An interactive experience made in Unity inspired by the painting "Umezawa Manor in Sagami Province" by Katsushika Hokusai. This project involves player exploration through an environment with modeled assets featuring item interaction such as rowing a boat with a spline based movement system.

SFML Platformer

A C++/SFML platformer game taking inspiration from Doodle Jump. This game includes simple enemies, infinite looping terrain to jump around and collectible items for scoring.
